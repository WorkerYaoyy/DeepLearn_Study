## 3.张量

**Tensor（张量）就是 PyTorch 里最基础的多维数组数据类型，用来存储数值数据、支持 GPU 加速和自动求导，是神经网络运算的核心载体。**

是一种代数对象

### 1. 按维度理解张量

- 0 维张量（标量 Scalar）：单个数字

  ```
  import torch
  t0 = torch.tensor(5)   # shape: ()
  ```

- 1 维张量（向量 Vector）：一维数组，一行 / 一列数据

  ```
  t1 = torch.tensor([1,2,3])  # shape: (3,)
  ```

- 2 维张量（矩阵 Matrix）：二维表格，常用作样本特征表

  ```
  t2 = torch.tensor([[1,2],[3,4]]) # shape: (2,2)
  ```

- 3 维张量：例如 `(通道, 高度, 宽度)` 的基础图片数据 (C, H, W)

- 4 维张量：例如批量图片 `(batch_size, 通道, 高度, 宽度)` —— `(N, C, H, W)`，深度学习最常用格式

- 更高维张量：用于时序、视频、3D 图像等复杂数据

### 2. 和普通 numpy 数组的区别

- ✅ 可以放到 **GPU 显卡**上运算，大幅加速深度学习计算
- ✅ 内置 **autograd 自动微分系统**：设置 `requires_grad=True` 后，可自动记录计算图、求梯度，做反向传播训练模型
- ✅ 配套 PyTorch 算子（卷积、矩阵乘法、激活函数等），适配深度学习框架生态
- ✅ 可以和 numpy 互相转换（`.numpy()` / `torch.from_numpy()`）

### 3. 关键属性

- `.shape` / `.size()`：查看各个维度尺寸
- `.dtype`：数据类型（float32、float64、int 等）
- `.device`：存储设备（CPU /cuda:0 等）
- `.requires_grad`：是否开启梯度计算
- `.data`：原始数值数据

### 4. 常见创建方式

```
# 直接创建
torch.tensor([[1,2],[3,4]])
# 全0、全1
torch.zeros(2,3)
torch.ones(2,3)
# 随机初始化
torch.rand(3,3)
```

### 5. 核心作用

整个 PyTorch 深度学习流程本质就是： **数据 → 转为 Tensor → 前向计算 → 自动求导更新参数 Tensor → 完成模型训练** 权重、输入图片、标签、梯度，全都是张量。

### 总结

- 本质：**支持 GPU + 自动求导的多维数组**
- 地位：PyTorch 的基础数据结构，神经网络运算的基本单元
- 区别：不止是数组，是专为深度学习优化的数据容器

#### 梯度（Gradient，符号：∇f / gradf）

梯度是**多元标量函数由一阶偏导数构成的向量**，用来描述标量场在某一点的变化率与最优变化方向

- 二元函数 f(x,y)：

∇f=∂x∂fi+∂y∂fj

- 三元函数 f(x,y,z)：

∇f=∂x∂fi+∂y∂fj+∂z∂fk

- n 元函数 f(x1,x2,...,xn)：

∇f=(∂x1∂f,∂x2∂f,…,∂xn∂f)

∇ 叫 **Nabla（哈密顿）算子**：∇=(∂x∂,∂y∂,∂z∂

总结：梯度是偏导数拼成的向量：指向函数增长最快方向，模长是最大变化率，垂直于等值面；负梯度就是最快下降方向。
